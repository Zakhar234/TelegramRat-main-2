# TelegramRat
**TelegramRat - Telegram Rat интегрированный прямо в бота телеграм!**

:writing_hand: **Инструкция по использованию:**
1.Сначала распакуйте архив
2.Скачайте обязательно python 3.12 с официального сайта https://www.python.org/downloads/release/python-3120/
Все .py файлы открывайте через ЛКМ потом ПКМ и наводите курсор мыши на EDIT WITH IDLE, а затем EDIT WITH IDLE 3.12(64-bit)
3.Скачайте Visual C++ с официального сайта https://visualstudio.microsoft.com/visual-cpp-build-tools/
Нажмите скачать "Download Build Tools" после загрузки выберите "Desktop development with C++" (Разработка классических приложений на C++).
ОБЯЗАТЕЛЬНО ПЕРЕЗАГРУЗИТЕ ПК
4.Нажмите пуск и найдите DEVELOPER COMMAND PROMPT откройте 
и ведите поочерёдно эти команды:
   1)where cl
   2)pip install aiogram==2.25.1
   3)pip install aiohttp==3.9.0
   4)pip install Babel==2.9.1
5.Проверьте там же
pip show aiogram
pip show aiohttp
pip show Babel
Должно быть:

aiogram: 2.25.1 

aiohttp: 3.9.0 

Babel: 2.9.1 
6.Сделайте бота в @BotFather если не знает как идите смотрите в ютуб.
После создания скопируйте токен бота который получили вот пример 1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
И вставьте его в config.py место примера
7.Узнайте свой id телеграма у бота @myidbot через команду /getid и тоже скопируйте и вставьте в  config.py
вместо примера
8.После того как вы сделали всё что я сказал вам выше запускайте compile.bat потом ждите после должно выйти 
"Файл создан"и типа того и закрывайте консоль main будет в папке dist и забыл сказать папки которые создаются 
во время создания main сами удалятся их не трогайте
9.Всё ваш вирус готов к использованию можете протестить на себе
